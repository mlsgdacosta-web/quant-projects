# Quantitative Research Projects  
*A collection of simulations, numerical methods, and probability work showcasing quantitative skills relevant to trading, research, and algorithmic modeling.*

This repository contains a suite of quantitative projects built in Python.  
They cover **stochastic processes**, **market microstructure**, **mechanism design**, **optimization**, and **time-series modeling**

#  Project Overviews

## **1. Monte Carlo Simulation Engine**
Implements:
- Geometric Brownian Motion (GBM)  
- Ornstein–Uhlenbeck process  
- Jump-diffusion models  
- Markov chains  
- Variance reduction (antithetic variates, stratified sampling)

**Skills:**  
stochastic calculus, simulation, numerical stability, vectorized computation

---

## **2. Market Maker Simulation (Limit Order Book)**
Event-driven limit order book model with:
- synthetic order flow  
- inventory-based market-making strategy  
- spread adjustment  
- PnL + inventory penalty  

**Skills:**  
market microstructure, execution, Poisson processes, trading strategy evaluation

---

## **3. Mechanism Design Optimization**
Solves:
- nonlinear pricing models  
- capacity-constraint problems  
- grid search, gradient descent, dynamic programming  
- profit maximization under IC/IR conditions  

**Skills:**  
optimization, numerical methods, microeconomic theory, algorithmic modeling

---

## **4. Time-Series Forecasting (ARIMA & GARCH)**
Includes lightweight implementations of:
- ARIMA  
- GARCH(1,1)  
- volatility clustering  
- rolling-window forecasting  

**Skills:**  
financial econometrics, volatility modeling, forecasting error analysis

---

## **5. Probability Problem Archive**
40+ solved problems covering:
- Markov chains  
- random walks  
- martingales  
- conditional expectation  
- combinatorics  
- variance/covariance puzzles  

With **Python simulations** verifying results.

**Skills:**  
mathematical reasoning, proof techniques, simulation for verification
